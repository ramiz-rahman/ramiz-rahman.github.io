Test Your Twitch

This is a simple game that measures the reaction time of the player. Upon pressing the start button, a shape will appear in the empty box and the player has to click on the shape as soon as possible. Every time (s)he clicks the shape, the time is taken to do so, aka his reaction time is measured. After ten clicks, the game ends and the player is shown some statistics, namely his best, worst and average reaction times.

The web page that houses the game is written entirely in HTML and styled using CSS. The game mechanics is implemented using client-side JavaScript. There is no backend or a database connection.

Both design and code for this project were done by me.
